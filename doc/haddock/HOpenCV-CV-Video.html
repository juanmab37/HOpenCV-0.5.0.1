createVideoWriter :: String -> FourCC -> Double -> CvSize -> IO (ErrCV CvVideoWriter)

Create a video file writer. The parameters are the file name, the 4-character code (of the codec used to compress the frames (e.g. ('F','M','P','4') for MPEG-4), the framerate of the created video stream, and the size of the video frames. The returned action may be used to add frames to the video stream.
